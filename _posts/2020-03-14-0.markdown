알고리즘 Problem Solving을 연습해야겠다는 생각이 들어서 공부를 제대로 하기 시작했다.
먼저 코딩 테스트 환경에 맞춰 개발환경을 세팅했다.

```c
const Razorpay = require('razorpay');

let rzp = Razorpay({
	key_id: 'KEY_ID',
	secret: 'name'
});

// capture request
rzp.capture(payment_id, cost)
	.then(function (data) {
		return 2;
	})
```
